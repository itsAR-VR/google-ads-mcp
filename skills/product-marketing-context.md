# product-marketing-context

## Description

Shared context skill for building the messaging layer behind campaigns, ads, and landing pages.
This skill is the upstream memory that helps every other workflow avoid rediscovering the same ICP, pain points, proof, differentiators, and offer language every time.

## When to use

Use when the user wants to:
- define or refresh ICP and positioning context
- improve ad-message consistency
- align landing pages with campaign intent
- create a reusable paid-acquisition context document

## Required inputs

- product or service overview
- ICP or audience segments
- category and competitors
- pains, desires, objections, differentiators
- proof points, testimonials, guarantees, pricing cues

## Steps

1. Define ICP segments and buying intent.
2. Capture pains, desired outcomes, objections, and triggers.
3. Map differentiators against competitors.
4. Capture offer components and proof assets.
5. Produce a reusable context document for downstream skills.

## Output format

- ICP summary
- pains and desired outcomes
- objection map
- differentiators
- offer framing
- proof bank
- words to use / avoid
- priority messaging angles
